# RealTimeChat

RealTimeChat is a learning project for building a real-time chat application using React, ASP.NET Core, and SignalR.

The project is designed to help explore real-time messaging, full-stack application structure, and clean architecture principles.

## Tech Stack

- **Frontend:** React
- **Backend:** ASP.NET Core
- **Real-time Communication:** SignalR

## Architecture

The solution is organised into clear layers:

- **Hubs** for real-time transport and client communication
- **Application services** for use-case logic
- **Domain models** for core business objects
- **In-memory repository** for development-time storage

## Purpose

This project is focused on learning:

- how real-time communication works in practice
- how to structure a full-stack application cleanly
- how to separate transport, business logic, and domain concerns

## Status

This project is currently under active development as part of a step-by-step learning process.
